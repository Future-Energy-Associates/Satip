    User Inputs¶

    We have to specify the bucket where the data is located

    zarr_bucket = 'solar-pv-nowcasting-data/satellite/EUMETSAT/SEVIRI_RSS/full_extent_TM_int16'


    Loading Data¶

    ds = io.load_from_zarr_bucket(zarr_bucket)
